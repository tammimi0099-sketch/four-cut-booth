## 🚀 How to Run Locally

1. Download or clone this repository to your computer.
2. Ensure your custom frame is saved as `assets/s1.png`.
3. Open `index.html` in any modern web browser (Chrome, Safari, Edge, Firefox).
4. **Note:** Web browsers strictly require a secure connection (`https://` or `localhost`) to grant webcam access.
